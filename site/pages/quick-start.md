For the full PreTeXt experience, you will likely want to either run PreTeXt locally on your computer, or use a cloud-based development environment such as GitHub Codespaces.

### Codespace Setup

You can use the full set of PreTeXt tools in a GitHub Codespace, which is a cloud-based development environment that runs in your browser. You can use it for free with a GitHub account, and it is a great way to get started without installing anything on your computer.

- [PreTeXt Codespace Template](https://github.com/pretextbook/pretext-codespace)
- [Getting Started Tutorial for Codespaces](https://pretextbook.org/doc/guide/html/tutorial.html)

### Local Setup

If you prefer to run PreTeXt locally on your own computer, you can follow the instructions in the [PreTeXt Guide](https://pretextbook.org/doc/guide/html/tutorial-install.html) to install the PreTeXt CLI and tools, either using a Docker container or installing Python packages directly.

### Tutorials for Instructors

If you are interested in using PreTeXt for course materials, checkout [PreTeXt for Instructors](https://pfi.mathtech.org/) for a tutorial and demo course you can use as a template.
